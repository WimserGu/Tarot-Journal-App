- Deck: Rider–Waite–Smith Tarot, “Pam-A” scan set
- Illustrator: Pamela Colman Smith
- Design: A. E. Waite
- Original publication: 1909; the selected Commons files are described as 1910 images
- Source collection: Wikimedia Commons, `Rider–Waite–Smith tarot deck (TaionWC)`
- Source URL: https://commons.wikimedia.org/wiki/Category:Rider-Waite-Smith_tarot_deck_(TaionWC)
- Accessed: 2026-07-14
- Front assets included: 78 of 78
- License label: Public Domain Mark, verified through each file's Commons API metadata

## Source and reuse boundary

The selected Commons category identifies itself as a same-style set of 78 scans of the “Pam-A” Rider–Waite–Smith deck. Each included file returned `Public domain` for both `LicenseShortName` and `UsageTerms` when audited through the Wikimedia Commons API. `source-files.json` records every source filename, description page, original URL, Commons SHA-1, source dimensions, and license fields.

The original illustrations are historical RWS artwork, not AI-generated replacements and not images copied from a modern commercial deck. The project does not include U.S. Games editions, Centennial Edition scans, Radiant Rider–Waite, Universal Waite, Albano–Waite, retail images, or other modern recolorings.

Public-domain status varies by jurisdiction and can be affected by local copyright terms. Wikimedia Commons states the individual files are public domain in their source country and the United States. Modern restorations, recolorings, typography, packaging, and card backs may carry separate rights; they are outside this asset set.

## Processing

The build script downloads a 700-pixel-wide Commons derivative of each verified source file, applies EXIF orientation, converts to RGB, scales with aspect ratio preserved, centers it on a neutral canvas, and saves an optimized progressive JPEG at 456 × 787 pixels. It does not crop symbolic content, recolor, apply saturation filters, redraw imagery, or replace titles. This single optimized size is used for table and Focus views; the card river loads only the repeated card-back image.

## Card back and fallback

`back/rws_back.png` is original Tarot Journal App artwork generated by `scripts/fetch_rws_assets.py`. It uses a simple rotationally symmetrical geometric pattern and does not copy a modern commercial Rider card back.

`fallback_front.png` is an original neutral framed placeholder generated by the same script. It contains no hidden identity or RWS imagery.

## Reproduction

Run `scripts/fetch_rws_assets.py` with the bundled Python/Pillow runtime. The script refuses any file that is missing or whose Commons metadata no longer contains a public-domain label. It also regenerates `source-files.json`, the normalized files, card back, fallback, and the centralized static Expo require map.

No image binary, image URL, or asset path is stored in DrawSession, Reading, AsyncStorage, or Supabase. Historical records resolve artwork from their existing stable `tarotCardId`.
